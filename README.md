# Experiment 2: Pattern Printing (Stars)

## Problem Statement
In this task, you have to print $n$ stars (`*`) vertically and $n$ horizontally.

## Input Format
The first line of input contains an integer $n$.

## Output Format
Print the stars in horizontal and vertical directions.

### Example 1
**Input:**
```text
3
```

**Output:**
```text
* * *
*
*
*
```

**Explanation:**

The output consists of a horizontal line of $n$ stars separated by spaces, followed by $n$ lines where each line contains a single star. Note that the first star of the vertical line essentially starts a new line after the horizontal row.
